# SeniorDev-game-CLI
A game CLI base with Python that let you fell how to be a SeniorDev

# 💻 Senior Developer Simulator (CLI Game)

Game berbasis teks sederhana yang disimulasikan langsung lewat terminal. Anda berperan sebagai Senior Developer yang harus bertahan hidup di bawah tekanan deadline proyek selama 5 hari!

## 🚀 Fitur Game
- Sistem manajemen status (`Sanity`, `Caffeine`, `Code Quality`, `Progress`).
- Skenario interaktif harian khas dunia kerja IT.
- Efek teks mengetik lambat (*typing effect*) di terminal.

## 🛠️ Cara Menjalankan
1. Pastikan Anda sudah menginstal Python 3.
2. Clone repo ini: `git clone https://github.com`
3. Masuk ke direktori: `cd senior-developer-simulator`
4. Jalankan game: `python app.py`
